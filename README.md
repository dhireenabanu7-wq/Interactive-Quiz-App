# 🎯 Interactive Quiz Application (Capstone Project)

A responsive Quiz Application built using HTML, CSS, and Vanilla JavaScript.

This project is part of the Capstone Program under Track A (Dev Agency Workflow) and demonstrates DOM manipulation, state management, and localStorage usage.

---

## ⚠️ Disclaimer

This is a frontend-only project made for learning purposes.

- No backend used
- No real user data collected
- All data stored in browser localStorage

---

## 🎯 Features

- Category-based quiz (HTML / CSS / JavaScript)
- Difficulty levels (Easy / Medium / Hard)
- Timer-based questions (15 seconds)
- Score tracking (+10 per correct answer)
- Accuracy calculation
- Leaderboard (Top 5 scores stored in localStorage)
- Dark / Light mode 
- Responsive design

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- localStorage

## 🚀 How to Run

1. Clone the repo
   git clone <repo-link>

2. Go to folder
   cd quiz-app

3. Open index.html in browser

---

## 🧠 How It Works

- User selects category and difficulty
- Quiz starts with timer
- Each correct answer gives +10 points
- Wrong or timeout reduces performance
- Final result shows score, accuracy, and leaderboard

---

## 📊 Scoring System

- Correct Answer = +10
- Wrong Answer = 0
- Timeout = Wrong
